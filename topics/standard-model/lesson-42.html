自發對稱破缺：墨西哥帽位能完整推導

學習目標

  • 完整推導「墨西哥帽」位能的簡併真空結構
  • 理解「自發」對稱破缺的核心概念：拉格朗日量保持對稱，物理真空態卻自發選擇一個方向
  • 建立圍繞選定真空的場漲落參數化，為第 43 課鋪路

第 41 課證明了希格斯玻色子的存在是理論的必然要求。這一課要正式 介紹它的數學起源——自發對稱破缺，一個看似簡單、卻是整個 標準模型質量起源的核心機制。

1. 一個帶錯誤符號質量項的複純量場

考慮第 29 課用過的複純量場 \(\phi\)，但把位能項的符號故意反轉：

\[ V(\phi) = -\mu^2\phi^*\phi + \lambda(\phi^*\phi)^2, \qquad \mu^2>0,\ \lambda>0 \]

這個位能在整體 U(1) 相位變換 \(\phi\to e^{i\alpha}\phi\) 下仍然不變 （只依賴 \(|\phi|\)，第 29 課的對稱性完整保留），但第一項的係數 是負的——第 20 課推導 Klein–Gordon 方程式時， 質量項的係數本應是正的（\(+m^2c^2/\hbar^2\)），這裡故意選一個 「錯誤符號」的質量項，接下來會看到這個選擇帶來的戲劇性後果。

2. 完整推導：簡併的真空

找這個位能的極值。設 \(\rho\equiv|\phi|\)，位能只依賴 \(\rho\)： \(V(\rho)=-\mu^2\rho^2+\lambda\rho^4\)。對 \(\rho\) 微分求極值：

\[ \frac{dV}{d\rho} = -2\mu^2\rho + 4\lambda\rho^3 = 2\rho(2\lambda\rho^2-\mu^2) = 0 \]

兩個解：\(\rho=0\) 或 \(\rho^2=\mu^2/(2\lambda)\)。判斷哪個是真正的最低點，看二階導數：

\[ \frac{d^2V}{d\rho^2}\bigg|_{\rho=0} = -2\mu^2 < 0 \quad(\text{局部極大值，不穩定}) \]

真空是一整圈簡併的最低點

\(\rho=0\)（也就是 \(\phi=0\)，原本「顯然」的真空選擇） 竟然是一個不穩定的局部極大值！真正的最低點在： \[ \rho = \rho_{\min} \equiv \frac{v}{\sqrt2}, \qquad v\equiv\sqrt{\frac{\mu^2}{\lambda}} \] （\(v\) 是標準記號，稱為真空期望值，稍後會反覆出現。） 因為位能只依賴 \(\rho=|\phi|\)、不依賴相位角， 滿足 \(|\phi|=v/\sqrt2\) 的所有 \(\phi\) 值——也就是複平面上 一整個半徑為 \(v/\sqrt2\) 的圓——都是能量相同、 同樣是最低點的簡併真空。這個位能畫出來的形狀 （縱軸能量、橫軸複數 \(\phi\) 平面）像一頂墨西哥帽： 中央有一個隆起的不穩定尖峰，四周是一圈等高的谷底。

3. 「自發」對稱破缺的核心概念

這是整個機制裡最深刻的一點。拉格朗日量本身——包括這個位能 \(V(\phi)\)——完整保留 U(1) 對稱性：整個「墨西哥帽」形狀本身， 繞著中心軸旋轉是完全對稱的。但物理系統必須選擇一個 具體的真空態來實際存在——量子系統的基態只能是圓上某一個特定的點， 不能同時「是」圓上所有點的疊加（這牽涉到真空態的物理本質， 與模組 C 討論的疊加原理不同，這裡不深入其量子場論的技術細節， 但結論是：物理上被實現的真空，就是圓上一個確定的點）。

對稱性「隱藏」而非「消失」

一旦系統選定了圓上的某一個點作為真空（習慣上選擇 \(\phi_0=v/\sqrt2\)，即相位角為零的那一點），繞著這個真空點 做微小擾動、觀察物理現象時，U(1) 對稱性看起來就「不見了」—— 因為圓上其他方向的點，能量雖然相同，卻是物理上不同的態。 這就是「自發對稱破缺」名稱的精確含義：對稱性沒有從方程式裡 消失（拉格朗日量仍然完整保有 U(1) 對稱），只是系統選擇了 一個特定的真空、把這個對稱性隱藏了起來，不再從圍繞這個真空的 物理現象裡直接看得出來。

4. 圍繞選定真空的場漲落參數化

選定真空 \(\phi_0=v/\sqrt2\) 之後，把場改寫成「真空值加上漲落」的形式， 用一個能同時捕捉「徑向漲落」與「角度漲落」的參數化：

\[ \phi(x) = \frac{1}{\sqrt2}\bigl(v+h(x)\bigr)e^{i\theta(x)/v} \]

（\(h(x)\) 是實數場，描述偏離真空半徑 \(v/\sqrt2\) 的徑向漲落； \(\theta(x)\) 是實數場，描述繞著圓周方向的角度漲落， 除以 \(v\) 是為了讓 \(\theta\) 本身帶有正確的能量因次。） 這個參數化的幾何意義很直接：\(h\) 描述「往山谷內外」的運動 （這個方向位能會變化，山谷兩側都在爬坡），\(\theta\) 描述 「沿著山谷底部」的運動（這個方向位能完全不變，因為整個圓周 都是簡併的最低點）。

檢查你的理解

  1. 能寫出墨西哥帽位能，並說明「錯誤符號質量項」這個選擇的意義
  2. 能完整推導真空期望值 \(v=\sqrt{\mu^2/\lambda}\)，並說明為什麼 \(\rho=0\) 是不穩定的
  3. 能準確說明「自發」對稱破缺的含義：對稱性是隱藏而非消失
  4. 能說明 \(h,\theta\) 兩個場各自對應墨西哥帽位能的哪個方向

下一課預告

第 43 課要把這個參數化代入完整的拉格朗日量，看 \(h\) 和 \(\theta\) 各自獲得什麼樣的質量——會發現 \(\theta\) 完全沒有質量 （Goldstone 定理），這正是第 44 課「規範玻色子吃掉 Goldstone 玻色子」這個關鍵機制的起點。
