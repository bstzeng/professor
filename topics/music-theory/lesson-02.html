模組 A｜音的原料：音名、音程與音階 · 第 2 課

音程（上）：度數與半音的算法

學習目標

  • 掌握「度數」的包含式數法：為什麼 C 到 E 是三度
  • 掌握「半音數」的精確計數：音程的第二把尺
  • 理解全音與半音，並用耳朵記住它們的差別

音程（interval）＝兩個音之間的距離。它是樂理的 「公分」：和弦是音程疊出來的（模組 C）、 旋律的情緒是音程走出來的（模組 E）。 麻煩的是，音樂用兩把不同的尺量這個距離—— 一把粗（度數）、一把細（半音數）——兩把都要會。

1. 第一把尺：度數（音名的距離）

包含式數法

度數只看音名字母（C D E F G A B）， 數的時候頭尾都算：
C → E：數 C、D、E，共 3 個字母 ⟹ 三度 （點聽 C–E）
C → G：C、D、E、F、G，共 5 個 ⟹ 五度 （點聽 C–G）
C → 高八度 C：8 個 ⟹ 八度—— 第 1 課「octave」（字源是拉丁文的「八」）的由來。
同一個音到自己：一度（不是零度！）。 這種「頭尾都算」的數法源自中世紀的傳統， 數學上不太優雅（一度＋一度不等於二度）， 但它已經深植所有樂譜、和弦記號與音樂溝通中， 必須照單全收。

2. 第二把尺：半音數（鍵盤的距離）

度數有個嚴重的盲點：它看不見黑鍵。C → E 是三度， C → E♭ 也是三度（字母都是 C 到 E），但兩者 聽起來明顯不同——先點 C–E、 再點 C–E♭ 比較一下（一個明亮、一個陰暗，第 3 課會正式命名 這個差別）。所以需要第二把尺：半音數—— 在鍵盤上從起點走到終點，經過幾個鍵（第 1 課： 相鄰兩鍵＝1 半音）。C → E 是 4 個半音， C → E♭ 是 3 個半音——第二把尺分得出來。

全音與半音

兩個最小的距離單位有專名：
半音（semitone）＝1 格，鍵盤上緊鄰的兩鍵： E–F、 C–C♯ ——聽起來緊、擠、帶著不安。
全音（whole tone）＝2 格： C–D、 F–G ——舒展、開闊得多。
輪流點幾次，把這兩種質感刻進耳朵—— 下一課的音階構造，全部用這兩塊積木搭。

3. 白鍵度數對照表（從 C 出發）

逐項列出（每一項都可點聽）

C–D：二度，2 半音
C–E：三度，4 半音
C–F：四度，5 半音
C–G：五度，7 半音
C–A：六度，9 半音
C–B：七度，11 半音
C–C：八度，12 半音
注意規律的「不規律」：度數每加一，半音數有時加 2 （跨全音）、有時只加 1（跨過 E–F 或 B–C 的天生半音， 第 1 課鍵盤圖上沒黑鍵的兩個位置）。 同樣的度數可以對應不同的半音數—— 這正是需要兩把尺的原因，也是下一課「大、小、完全」 這些形容詞的用武之地。

4. 給創作者的換算直覺

實務上兩把尺的分工：溝通用度數 （「往上三度和聲」「五度進行」——樂手之間的共通語言、 和弦記號的基礎），精算用半音 （移調、在樂器上找音、判斷兩個三度是「大」還是「小」）。 訓練方法很簡單：接下來每一課遇到音程，先猜半音數、 再點開聽——猜錯就重聽一次。兩週後你會發現 自己看到 C–A 直接反射出「六度、9 半音、 這個聲音」。

檢查你的理解（用耳朵）

  1. 能說明包含式數法：C 到 F 為什麼是四度、同音為什麼是一度
  2. 能不看表算出 C–B 的半音數，再點開驗證
  3. 先點 E–F 再點 F–G：閉著眼能分出哪個是半音嗎
  4. 能說出「同度數、不同半音數」的例子（C–E 與 C–E♭），並解釋為什麼需要兩把尺

下一課預告
