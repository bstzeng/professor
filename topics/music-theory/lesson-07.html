模組 B｜調性的地圖：五度圈 · 第 7 課

關係大小調與平行大小調：一張調號、兩種心情

學習目標

  • 掌握關係大小調的規律：小三度下方、共用調號
  • 掌握平行大小調：同主音、不同調號的色彩對比
  • 理解兩種關係各自在創作裡的典型用法

第 5 課發現 C 大調與 A 小調用完全相同的七個音 ——只是主音不同。這不是特例：每個大調都有一個 這樣的「室友」。這一課把兩種大小調配對關係 （共用音的、共用主音的）一次理清楚—— 它們是創作裡操縱明暗最基本的兩個開關。

1. 關係大小調：共用鑰匙的室友

規律與推導

規律：每個大調的第 6̂ 音， 就是它關係小調的主音——換算音程：主音往下 小三度（3 個半音）。
C 大調（第 6̂＝A）↔ A 小調； G 大調（第 6̂＝E）↔ E 小調； F 大調 ↔ D 小調……
為什麼恰好是第 6̂：從大調第 6̂ 出發 走完七個音，間距序列恰好重排成 「全半全全半全全」——自然小調的公式（第 5 課 的驗算就是這件事）。共用全部七個音 ⟹ 共用同一張調號：看到一個升記號， 可能是 G 大調、也可能是 E 小調——分辨靠聽 主音是誰（第 9 課的主題）。
試聽室友關係： G 大調 與 E 小調 ——同一批音、兩張面孔。

2. 平行大小調：同一個家、換掉家具

同主音的明暗開關

平行大小調＝同一個主音的大調與小調： C 大調 ↔ C 小調。主音相同，但成員大搬風—— C 小調要三個降記號（E♭、A♭、B♭）， 與 C 大調差了三個音（在五度圈上，平行小調的調號 永遠在大調逆時針三步處——關係調差 0 步、 平行調差 3 步，這兩個數字記住，第 8 課量距離 直接用）。
平行調的威力是最直接的明暗對比—— 主音不動、第 3̂ 降半音（大三度→小三度， 第 3 課的招牌色對調）。聽這組對比： C 大三和弦 vs C 小三和弦； 再聽整條音階： C 大調 vs C 小調 ——像同一個房間開燈與關燈。

3. 創作裡的用法

兩個開關、兩種效果

關係調切換＝溫柔的轉場：因為音完全共用， 從 C 大調滑進 A 小調毫無阻力——大量流行歌的 主歌偏 vi（小調感）、副歌回 I（大調感）， 用的正是這個無縫通道。聽這個對比： 偏 Am 的進行（主歌感） vs 偏 C 的進行（副歌感） ——同樣四個和弦，重心不同，明暗就不同 （這兩組進行的完整解剖在第 17 課）。
平行調切換＝戲劇性的變色：主音不動、 色彩驟變，適合段落之間的強烈反差 （例如小調主歌 → 大調副歌的「撥雲見日」）。 聽 C 小調樂句 直接接 C 大調樂句 ——這種手法在模組 F 第 28 課（借用和弦）會升級成 更細膩的版本：不整段切換，只「借」一兩個和弦。

檢查你的理解（用耳朵）

  1. 能說出任一大調的關係小調怎麼找（第 6̂ 音／下方小三度），並舉三組例子
  2. 能說明為什麼關係調共用調號、平行調的調號差三步
  3. 點 C 與 Cm 和弦，能說出差別集中在哪一個音級嗎
  4. 能各說出關係調與平行調在段落安排上的一種典型用法
