Schwarzschild 時空中的測地線：軌道與水星近日點進動

學習目標

  • 利用 Schwarzschild 時空的基靈向量，寫下類時測地線沿途守恆的能量與角動量
  • 用四維速度歸一化條件推導出徑向「能量方程式」，並整理成類牛頓形式的有效位能
  • 指出相對論有效位能中純粹是廣義相對論效應的 \(1/r^3\) 修正項，理解它為什麼導致軌道進動
  • 知道水星近日點進動公式 \(\Delta\varphi=6\pi GM/(c^2a(1-e^2))\) 及其歷史意義

第 25、26 課建立了 Schwarzschild 度規並釐清了它的幾何結構。這一課要問一個非常實際的問題： 在這個時空中，有質量的物體（例如行星）究竟怎麼運動？答案就是沿著類時測地線運動—— 第 17 課已經確立這是廣義相對論裡「自由落體」的精確意義。這一課要具體求解這些測地線， 並用結果解釋廣義相對論最早、也最有名的觀測驗證之一：水星軌道的反常進動。

1. 化簡到赤道面，寫下守恆量

由於 Schwarzschild 時空是球對稱的，任何一條測地線的角動量都定義了一個固定的平面 （正如牛頓重力下的克卜勒軌道一樣），不失一般性可以選取座標，使這個平面正好是 \(\theta=\pi/2\) （赤道面）。整個問題於是化簡成二維：只需要追蹤 \(r(\tau)\) 與 \(\varphi(\tau)\) 隨原時的變化。

Schwarzschild 度規具有兩個明顯的基靈向量：時間平移 \(\xi_{(t)}^\mu=\partial_t\)（因為度規分量不顯式依賴 \(t\)）與繞對稱軸旋轉 \(\xi_{(\varphi)}^\mu=\partial_\varphi\)（不顯式依賴 \(\varphi\)）。 依第 24 課的結果 \(E=-g_{\mu\nu}\xi^\mu u^\nu=\text{常數}\)，這兩個基靈向量分別給出沿測地線守恆的 「單位質量能量」與「單位質量角動量」：

Schwarzschild 軌道的守恆量

\[ \frac{E}{c} = \left(1-\frac{2GM}{c^2 r}\right)c\,\frac{dt}{d\tau}, \qquad L = r^2\frac{d\varphi}{d\tau} \]

\(E\) 是粒子的守恆能量（含靜質量能，遠處 \(r\to\infty\) 時 \(E\to\gamma mc^2\) 的形式）， \(L\) 是繞對稱軸的守恆角動量，兩者都對每單位靜止質量而言，與牛頓力學中能量、角動量守恆的角色相對應， 但這裡是嚴格從度規的對稱性（基靈向量）推導出來，不是額外假設。

2. 徑向「能量方程式」與有效位能

除了上面兩個守恆量，四維速度還必須滿足歸一化條件（第 5 課）\(u^\mu u_\mu = -c^2\)。 在赤道面上展開（\(\theta=\pi/2\) 使 \(d\theta=0\)）：

\[ -\left(1-\frac{2GM}{c^2r}\right)c^2\left(\frac{dt}{d\tau}\right)^2 + \left(1-\frac{2GM}{c^2r}\right)^{-1}\left(\frac{dr}{d\tau}\right)^2 + r^2\left(\frac{d\varphi}{d\tau}\right)^2 = -c^2 \]

把上一節的 \(dt/d\tau\) 與 \(d\varphi/d\tau\) 代入替換，兩邊乘上 \((1-2GM/c^2r)\)，經過整理（過程只是代數運算，不再需要新的物理輸入），可以解出：

\[ \left(\frac{dr}{d\tau}\right)^2 = \frac{E^2}{c^2} - c^2\left(1-\frac{2GM}{c^2r}\right)\left(1+\frac{L^2}{c^2r^2}\right) \]

把右邊展開、重新分組成「牛頓形式」\(\frac12(dr/d\tau)^2 + V_{\text{eff}}(r) = \text{常數}\) （常數取為 \((E^2-m^2c^4)/2mc^2\) 型的量，這裡取單位質量所以省略 \(m\)），得到：

Schwarzschild 有效位能

\[ \frac12\left(\frac{dr}{d\tau}\right)^2 + V_{\text{eff}}(r) = \frac{E^2-c^4}{2c^2}, \qquad V_{\text{eff}}(r) = -\frac{GM}{r} + \frac{L^2}{2r^2} - \frac{GML^2}{c^2 r^3} \]

前兩項 \(-GM/r + L^2/(2r^2)\) 正是牛頓力學中克卜勒問題的有效位能——重力位加離心位障。 第三項 \(-GML^2/(c^2r^3)\) 在牛頓極限（\(c\to\infty\)）中消失，是純粹相對論性 的修正項：它是負的、且在小 \(r\) 時（\(1/r^3\) 比 \(1/r^2\) 增長更快）會壓過離心位障項， 使得有效位能在足夠小的 \(r\) 不再有離心力阻擋粒子繼續墜落——這是牛頓重力完全沒有的行為。

3. 為什麼多出的 \(1/r^3\) 項會造成軌道進動

在純牛頓的有效位能 \(-GM/r+L^2/(2r^2)\) 下，束縛軌道（\(E\) 適當為負的情形）恰好是閉合 的橢圓——這是克卜勒問題一個特殊的、不平凡的性質（源自牛頓重力 \(1/r^2\) 力這個特定冪次背後隱藏的 額外對稱性，稱為 Laplace–Runge–Lenz 對稱）。只要在位能中加入任何其他冪次的修正項—— 像 Schwarzschild 有效位能裡的 \(-GML^2/(c^2r^3)\)——這個特殊的閉合性質就被破壞。

物理圖像是：粒子仍然在近日點（最靠近中心的點）與遠日點（最遠的點）之間來回振盪， 但每完成一次徑向振盪（從近日點回到近日點）所掃過的角度 \(\Delta\varphi\)，不再精確等於 \(2\pi\)，而是 \(2\pi\) 加上一個小的修正量。這代表橢圓軌道的長軸本身會緩慢地繞中心天體 旋轉（進動），軌道本身永遠不會精確重複同一條路徑——就像一朵緩慢轉動的玫瑰花瓣圖案。

4. 近日點進動公式與水星

對近乎圓形（\(1/r^3\) 項是小修正）的橢圓軌道，半長軸為 \(a\)、離心率為 \(e\)， 對有效位能做微擾展開可以求出每繞行一圈、近日點角度的淨進動量：

近日點進動公式

\[ \Delta\varphi = \frac{6\pi GM}{c^2 a(1-e^2)} \quad \text{（每繞一圈）} \]

這個公式歷史意義非凡。19 世紀天文學家勒維耶（Le Verrier）發現水星近日點每世紀進動約 5600 角秒， 扣除其他行星重力攝動的貢獻後，仍留下約每世紀 43 角秒的反常進動，牛頓重力完全無法解釋 （曾有人假設存在一顆未發現的行星「祝融星」來解釋，但始終找不到）。愛因斯坦在 1915 年 11 月用剛完成的 廣義相對論代入水星的軌道參數（\(a\approx 5.79\times10^{10}\,\text{m}\)，\(e\approx0.206\)， \(M\) 取太陽質量），算出的 \(\Delta\varphi\) 換算成每世紀恰好是約 43 角秒—— 與觀測完美吻合，不需要任何自由參數的調整。這是廣義相對論最早、也是最具說服力的觀測驗證之一， 據說愛因斯坦本人事後形容當時「心臟砰砰跳了好幾天」。

檢查你的理解

  1. Schwarzschild 度規的哪兩個基靈向量分別給出守恆能量 \(E\) 與守恆角動量 \(L\)？它們各自對應什麼樣的座標對稱性？
  2. 有效位能中的 \(-GML^2/(c^2r^3)\) 項在牛頓極限下如何消失？它與離心位障項 \(L^2/(2r^2)\) 在小 \(r\) 時的競爭如何影響粒子的命運？
  3. 為什麼牛頓重力下的束縛軌道是精確閉合的橢圓，而 Schwarzschild 時空中的軌道不是？這與位能的冪次結構有什麼關係？
  4. 水星近日點反常進動的觀測值大約是多少？廣義相對論的預測為什麼被認為是一個特別有說服力的驗證（提示：有沒有自由參數可以調整）？
