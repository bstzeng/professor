模組 E｜光譜學：原子內部結構的第一道線索 · 第 25 課

巴耳末公式的發現：氫原子光譜的神祕規律

學習目標

  • 認識氫原子在可見光範圍內的四條譜線（Hα、Hβ、Hγ、Hδ）
  • 完整驗證巴耳末公式，用它精確重現四條線的波長
  • 理解這個公式的性質：純粹的數字擬合，沒有任何物理推導支撐

模組 E 走到這裡，光譜學已經證明自己能鑑定成分（第 23 課）、 能發現新元素（第 24 課）。這一課要看它面對最簡單 的元素——氫——時，揭露出一個更深、更奇怪的謎團。

1. 氫的四條可見光譜線

氫是元素中最簡單的一個（第 19 課提過，氫的原子量被普勞特 當成基本單位）。把氫氣激發到發光狀態、用稜鏡展開它的光譜 （第 23 課的發射光譜技術），會在可見光範圍看到四條 分立的亮線，由紅到紫依序命名為 \(H_\alpha, H_\beta, H_\gamma, H_\delta\)：

實測波長（單位：奈米 nm）

\(H_\alpha\)（紅）\(= 656.3\)
\(H_\beta\)（藍綠）\(= 486.1\)
\(H_\gamma\)（藍紫）\(= 434.0\)
\(H_\delta\)（紫）\(= 410.2\)

這四個數字乍看毫無規律——不是等差數列，也不是簡單的 倍數關係。它們就這樣被測量、記錄了幾十年， 沒有人看出裡面藏著什麼。

2. 巴耳末：一位六十歲教師的數字遊戲

1885 年，瑞士巴塞爾一所女子中學的數學教師巴耳末 （Johann Balmer，時年 60 歲，並非專業物理學家或化學家） 對這四個波長做了一件純粹的數字實驗：不問「為什麼」， 只問「這四個數字能不能被一個簡單的公式套進去」。 他嘗試了各種算術組合，最終找到了：

巴耳末公式

\[ \lambda = B \cdot \frac{n^2}{n^2 - 4}, \qquad n = 3, 4, 5, 6, \ldots \] 其中 \(B = 364.6\ \text{nm}\) 是一個由實驗數據反推出的常數， \(n\) 取大於 \(2\) 的整數。

3. 完整數值驗證

把 \(n = 3, 4, 5, 6\) 逐一代入，直接算出結果：

逐項核對

\(n=3\)： \[ \lambda = 364.6 \times \frac{9}{9-4} = 364.6 \times \frac{9}{5} = 656.3\ \text{nm} \] 與 \(H_\alpha\) 實測值 \(656.3\) 完全吻合 ✓

\(n=4\)： \[ \lambda = 364.6 \times \frac{16}{16-4} = 364.6 \times \frac{16}{12} = 486.1\ \text{nm} \] 與 \(H_\beta\) 實測值 \(486.1\) 完全吻合 ✓

\(n=5\)： \[ \lambda = 364.6 \times \frac{25}{25-4} = 364.6 \times \frac{25}{21} \approx 434.0\ \text{nm} \] 與 \(H_\gamma\) 實測值 \(434.0\) 完全吻合 ✓

\(n=6\)： \[ \lambda = 364.6 \times \frac{36}{36-4} = 364.6 \times \frac{36}{32} \approx 410.2\ \text{nm} \] 與 \(H_\delta\) 實測值 \(410.2\) 完全吻合 ✓

四條線，四個整數，一個常數 \(B\)——全部命中， 誤差落在測量精度之內。

4. 誠實面對：這不是推導，是擬合

純粹的經驗公式

必須非常誠實地說清楚這個公式的性質：巴耳末沒有 從任何物理原理出發推導出這個式子——他不知道氫原子內部 是什麼樣子，不知道 \(n\) 代表什麼物理量，甚至不知道 為什麼是 \(n^2\) 而不是別的形式。他純粹是拿著四個數字， 反覆嘗試代數形式，直到湊出一個公式能夠精確重現它們。 這與這門課先前建立的每一個推導（例如亞佛加厥常數的多重 獨立測量、第 21 課鎵的密度預言）性質完全不同——那些是 從理論推出可驗證的預言；巴耳末公式是反過來， 從已知數據硬湊出規律，完全沒有物理機制支撐。

但也正因為如此，這個公式的精確度才顯得更加不可思議—— 一個沒有任何理論根據的擬合，命中率卻高到不像巧合。 巴耳末自己也大膽推測，也許 \(n=3,4,5,6\) 之外的整數， 會對應到可見光範圍以外、當時還沒被觀測到的譜線。 下一課要看這個推測如何被推廣，並精確預言出當時 根本還沒被人看見的譜線系列。

檢查你的理解

  1. 能寫出氫原子四條可見光譜線的名稱與實測波長
  2. 能寫出巴耳末公式，並至少完整驗算其中一條譜線
  3. 能說明巴耳末公式與先前課程中「理論推導出預言」的方法有何本質不同
  4. 能說明為什麼一個純粹擬合出來的公式，命中率如此高依然令人驚訝
