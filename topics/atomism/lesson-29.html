模組 F｜古典物理的極限：黑體輻射危機 · 第 29 課

完整推導空腔內電磁波模式的計數

學習目標

  • 理解立方體空腔內，駐波邊界條件如何限制電磁波的波向量
  • 完整推導 k 空間裡「允許模式」的計數方式
  • 推導出模式密度函數 \(g(\nu)\)，為下一課的雷立—金斯公式鋪路

第 28 課用熱力學導出了黑體輻射的總量， 但要回答「每個頻率各佔多少能量」，必須先問一個更基本的 幾何問題：一個邊長 \(L\) 的立方體空腔，內壁是完美導體， 裡面能存在哪些電磁駐波？這一課只做純粹的計數， 還不涉及能量——計數完成後，下一課才會把能量放進去。

1. 邊界條件：駐波的量子化波向量

完美導體的內壁上，電場的切向分量必須為零（否則會驅動 無限大的表面電流）。這個邊界條件，逼迫腔內只能存在 駐波——就像兩端固定的弦只能振動特定的 波長一樣。沿 \(x\) 方向，波必須在 \(x=0\) 與 \(x=L\) 兩端都是節點，所以：

三個方向的量子化條件

\[ k_x = \frac{n_x \pi}{L},\quad k_y = \frac{n_y \pi}{L},\quad k_z = \frac{n_z \pi}{L}, \qquad n_x, n_y, n_z = 1, 2, 3, \ldots \] （\(n_x,n_y,n_z\) 必須是正整數—— 零或負整數不會產生新的、獨立的駐波模式。） 波向量的大小滿足 \[ k^2 = k_x^2+k_y^2+k_z^2 = \frac{\pi^2}{L^2}\left(n_x^2+n_y^2+n_z^2\right) \] 這與波長的關係是 \(k = 2\pi/\lambda\)，與頻率的關係是 \(k = 2\pi\nu/c\)（電磁波以光速 \(c\) 傳播）。

這個推導方式其實與模組 C（第 13 課氣體分子速度分布） 用過的策略遙相呼應：把一個連續問題（波的傳播）拆解成 離散的整數標籤 \((n_x,n_y,n_z)\)，再用計數的方式處理。

2. 建立「整數格點空間」

把每一組允許的 \((n_x,n_y,n_z)\) 看成三維空間裡的一個 格點——由於 \(n_x,n_y,n_z\) 都必須是正整數，這些格點 只填滿第一卦限（三個座標都為正的那個八分之一空間）。 格點之間的間距是 \(1\)（因為 \(n\) 每次增加 \(1\)）， 所以格點的密度是每單位體積 \(1\) 個。

3. 完整推導：計數半徑 \(n\) 到 \(n+dn\) 之間的模式數

球殼計數法

定義 \(n \equiv \sqrt{n_x^2+n_y^2+n_z^2}\)。在整數格點 空間裡，半徑 \(n\) 到 \(n+dn\) 之間的模式數，等於這個 薄球殼的體積（因為格點密度為 \(1\)），但只取第一卦限 （八分之一球殼，因為 \(n_x,n_y,n_z>0\)）： \[ dN = \frac{1}{8}\times 4\pi n^2\,dn = \frac{\pi n^2}{2}\,dn \] 再乘以電磁波的兩個獨立偏振方向 （橫波有兩個垂直於傳播方向的獨立振動方向， 這是電磁波作為橫波的基本性質）： \[ dN = 2\times\frac{\pi n^2}{2}\,dn = \pi n^2\,dn \]

4. 換成頻率變數：模式密度函數 \(g(\nu)\)

\(n\) 與頻率 \(\nu\) 的關係，可由 \(k=\pi n/L\) 與 \(k=2\pi\nu/c\) 消去 \(k\) 求得：

完整換元

\[ \frac{\pi n}{L} = \frac{2\pi\nu}{c} \quad\Longrightarrow\quad n = \frac{2L\nu}{c} \quad\Longrightarrow\quad dn = \frac{2L}{c}\,d\nu \] 代入 \(dN = \pi n^2\,dn\)： \[ dN = \pi\left(\frac{2L\nu}{c}\right)^2 \cdot \frac{2L}{c}\,d\nu = \pi\cdot\frac{4L^2\nu^2}{c^2}\cdot\frac{2L}{c}\,d\nu = \frac{8\pi L^3 \nu^2}{c^3}\,d\nu \] 由於空腔體積 \(V=L^3\)，單位體積的模式密度函數 （每單位頻率間隔、每單位體積內的駐波模式數）就是： \[ g(\nu)\,d\nu = \frac{dN}{V} = \frac{8\pi\nu^2}{c^3}\,d\nu \]

這是一個純粹幾何與邊界條件推出的結果，完全不涉及能量 或溫度——它只回答「有多少個獨立的容器」，還沒回答 「每個容器裡裝了多少能量」。下一課要把這個問題接上： 古典統計力學會告訴每個模式平均帶有多少能量， 兩者相乘，就是完整（但即將出問題）的黑體光譜公式。

檢查你的理解

  1. 能說明為什麼完美導體邊界會強迫腔內只能有駐波，且 \(n_x,n_y,n_z\) 必須是正整數
  2. 能完整推導出 \(dN=\pi n^2 dn\)，說明「1/8 球殼」與「乘以 2」兩個因子分別從何而來
  3. 能完整推導出 \(g(\nu)=8\pi\nu^2/c^3\)，並說出每一步換元用了什麼關係式
  4. 能說明模式密度函數 \(g(\nu)\) 本身還缺少哪個關鍵資訊，才能得到完整的能量光譜

下一課預告
