模組 F｜古典物理的極限：黑體輻射危機 · 第 28 課

熱輻射的古典理論：史特凡—波茲曼與維恩位移定律回顧

學習目標

  • 理解「黑體」的定義，並看見它與克希荷夫定律的直接關聯
  • 完整推導史特凡—波茲曼定律：熱力學 + 輻射壓力如何導出 \(u \propto T^4\)
  • 認識維恩位移定律的內容，理解它為模組 F 的核心問題埋下伏筆

模組 E 走到氫原子光譜的整數謎團（第 25、26 課）就誠實地 停下了。模組 F 要換一個看似完全無關的問題切入原子內部： 一塊燒得火紅的鐵、太陽表面的顏色——加熱的物體為什麼會 發出那個顏色的光？這條路線的終點， 會意外地產出物理史上第一個「量子」。

1. 黑體：一個理想化但可實現的模型

「黑體」（black body）指一個能完全吸收所有 照射到它身上的電磁輻射、不反射任何光的理想物體。 這聽起來像個純粹的假設，但克希荷夫（第 23 課已經介紹過他 的三條光譜定律）指出一個絕佳的實現方式：在一個空腔 （cavity，內壁不透明）上開一個小孔——任何進入小孔的光， 會在腔壁反覆吸收、反射，幾乎不可能再從小孔逃出， 所以這個小孔的表現就像完美的黑體。同一個空腔加熱後， 小孔也會發出光——這束光只由空腔的溫度決定， 與腔壁材質完全無關（這正是克希荷夫定律的推論：熱平衡下， 吸收與發射必須精確平衡，第 23 課定律三的邏輯在這裡 以另一種形式重現）。這門課接下來要問的問題就是： 這束「黑體輻射」的光譜——不同波長各自帶有多少能量—— 長什麼樣子？

2. 史特凡的實驗定律（1879）

奧地利物理學家史特凡（Josef Stefan）從實驗數據歸納出： 黑體單位面積、單位時間輻射出的總能量 \(j\)，正比於 絕對溫度的四次方：

史特凡定律（實驗歸納）

\[ j = \sigma T^4 \] 其中 \(\sigma\) 是純粹由實驗測得的常數。這是一個和 巴耳末公式（第 25 課）性質很像的起點——先有精確的 數字規律，理論解釋隨後才跟上。

3. 波茲曼的推導（1884）：熱力學 + 輻射壓力

五年後，波茲曼（Ludwig Boltzmann，第 13、17 課出現過的 統計力學奠基者）用純粹的熱力學論證，把這條經驗定律 推導出來——這是這一課唯一完整走過的推導， 引用一項來自馬克士威電磁理論的結果（該理論本身的建立 遠超這門課範圍，這裡只引用其結論）：填滿空腔的電磁輻射 會對腔壁施加輻射壓力 \(P\)，且 \[ P = \frac{u}{3} \] 其中 \(u = U/V\) 是輻射的能量密度。

完整推導：從熱力學馬克士威關係式到 \(u \propto T^4\)

對任意系統，熱力學第一、第二定律合寫成 \(dU = TdS - PdV\)，由此可以導出一個標準的馬克士威關係式： \[ \left(\frac{\partial U}{\partial V}\right)_T = T\left(\frac{\partial P}{\partial T}\right)_V - P \] 輻射場的能量只由溫度決定（\(U = uV\)，\(u=u(T)\)）， 所以左邊 \((\partial U/\partial V)_T = u\)。 代入 \(P = u/3\)，右邊的 \((\partial P/\partial T)_V = \tfrac{1}{3}\dfrac{du}{dT}\)。整條式子變成： \[ u = T\cdot\frac{1}{3}\frac{du}{dT} - \frac{u}{3} \] 整理： \[ u + \frac{u}{3} = \frac{T}{3}\frac{du}{dT} \quad\Longrightarrow\quad \frac{4u}{3} = \frac{T}{3}\frac{du}{dT} \quad\Longrightarrow\quad \frac{du}{u} = 4\,\frac{dT}{T} \] 兩邊積分：\(\ln u = 4\ln T + \text{const}\)，也就是 \[ u(T) = a\,T^4 \] 總輻射功率 \(j\) 正比於 \(u\)（能量密度乘以光速再除以 幾何因子），所以 \(j = \sigma T^4\)——史特凡的經驗定律， 從熱力學第一原理完整導出。這正是這門課從模組 C 以來反覆驗證的模式：先有現象，再有機制。

4. 維恩位移定律（1893）：峰值波長的規律

總功率只是黑體輻射故事的一半——史特凡—波茲曼定律沒有說 「哪個波長的光最亮」。德國物理學家維恩（Wilhelm Wien） 用另一套熱力學論證（涉及空腔緩慢絕熱壓縮時輻射如何 隨之改變，這個論證本身的技術細節超出這門課的工具， 這裡只引用其結論，不重複推導）得到：

維恩位移定律（結論引用）

\[ \lambda_{\max} T = b, \qquad b \approx 2.898\times10^{-3}\ \text{m·K} \] 溫度越高，輻射能量最集中的波長 \(\lambda_{\max}\) 就越短—— 這就是為什麼鐵被加熱到越高溫，顏色會從暗紅、經橘黃， 逐漸「藍移」。

5. 危機的種子：兩條定律都不是完整的光譜

史特凡—波茲曼定律給出總功率，維恩位移定律 只給出峰值波長的位置——兩者都沒有回答最根本 的問題：在任意波長 \(\lambda\)（或任意頻率 \(\nu\)）， 黑體到底輻射出多少能量？也就是完整的函數 \(u(\nu, T)\) 長什麼樣子？下一課要從最基本的電磁波邊界條件出發， 一步步逼近這個函數——而逼近的路上，會撞上古典物理 史上最著名的一次崩潰。

檢查你的理解

  1. 能說明黑體如何用開孔空腔實現，以及這與克希荷夫定律的關聯
  2. 能完整重做史特凡—波茲曼定律的推導：從 \(P=u/3\) 與馬克士威關係式到 \(u\propto T^4\)
  3. 能寫出維恩位移定律的內容與物理意義
  4. 能說明為什麼這兩條定律合起來還不足以回答「完整的光譜長什麼樣子」
