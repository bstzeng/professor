模組 A｜認識韓文字：訓民正音的智慧 · 第 3 課

子音的設計邏輯：模仿發音器官形狀的天才設計

學習目標

  • 理解五個基本子音的形狀如何對應發音器官的姿勢
  • 掌握「加筆畫＝加送氣」的規律，推導出其餘子音
  • 認識這套設計為什麼被語言學稱為「象形文字系統」

母音（第 2 課）的邏輯是幾何組合；子音的邏輯更驚人—— 每一個基本子音的形狀，直接畫的就是你發那個音時， 嘴巴裡的樣子。這是訓民正音最常被語言學界拿出來 稱讚的設計，語言學家把這種「字形直接對應發音動作」 的文字系統稱為「象形文字系統」（featural writing system）——韓文是舉世公認最典型的案例。

1. 五個基本子音：畫出發音器官

逐一對照

ㄱ（g/k）——畫的是舌根抬起、 抵住喉嚨後方的樣子（發 g/k 音時舌頭真的是這個姿勢）： 가（ga）。
ㄴ（n）——畫的是舌尖抵住上排牙齦的樣子： 나（na）。
ㅁ（m）——畫的是嘴唇閉合的方形輪廓： 마（ma）。
ㅅ（s）——畫的是牙齒（門齒）的形狀， 發 s 音時氣流從齒縫擠出： 사（sa）。
ㅇ（無聲或 ng）——畫的是喉嚨的圓形輪廓： 아（a，此時 ㅇ 不發音，只是佔位符——第 4 課會解釋這個規則）。
五個形狀，五個發音部位——不是抽象符號， 是簡化過的發音器官素描。

2. 完整推導：加筆畫代表加送氣

系統性的衍生規則

五個基本形狀確定後，訓民正音用「加一筆」 表示「發音時多送一口氣」，系統性地衍生出其餘子音：
ㄱ → ㅋ（k，送氣版）： 카（ka，比 가 更用力吐氣）
ㄴ → ㄷ → ㅌ（d → t，送氣版）： 다（da）、 타（ta）
ㅁ → ㅂ → ㅍ（b → p，送氣版）： 바（ba）、 파（pa）
ㅅ → ㅈ → ㅊ （s → j → ch，逐級加重）： 자（ja）、 차（cha）
ㅇ → ㅎ（h，喉音加送氣）： 하（ha）
規律一以貫之：筆畫越多，送氣越用力—— 記字形不再是死背，是理解一套加法規則。

3. 剩下兩個子音：ㄹ 與雙寫

還有一個常用子音 ㄹ（r/l 之間的音， 中文沒有精確對應，接近輕彈舌尖的音）： 라（ra）—— 它的形狀來自 ㄴ 的變形，代表舌尖更靈活的彈動。 十四個基本子音全部到齊。第 5 課會再看到， 把基本子音左右並排寫兩次會產生另一組 「硬音」（緊音），是這門課模組 B 的內容， 這裡先知道有這個延伸方式即可。

4. 為什麼這套設計特別好記

比較一下：學拉丁字母的「B」，你只能死記這個形狀 對應 /b/ 這個音，兩者沒有邏輯關聯。學韓文的 ㅁ，你同時記住「這是雙唇閉合的方形」， 形狀本身就是發音動作的提示——這正是第 1 課提到 「刻意設計成易學」的具體證據。下一課要看這些 母音與子音怎麼組合成一個個方塊字，也就是你在 韓國街頭招牌上真正會看到的樣子。

檢查你的理解（用耳朵）

  1. 能說出五個基本子音各自模仿的發音器官姿勢
  2. 能完整推導 ㄱ→ㅋ、ㅁ→ㅂ→ㅍ 的加筆畫規律
  3. 點聽 가 與 카，能感受到送氣的差別嗎
  4. 能說明為什麼這套設計比拉丁字母更容易記憶字形與發音的關聯
