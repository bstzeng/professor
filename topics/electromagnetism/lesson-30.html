模組 G｜馬克士威的統一 · 第 30 課

安培定律的矛盾：完整推導電荷守恆如何逼出位移電流

學習目標

  • 完整推導連續方程式：電荷守恆的微分形式
  • 用兩種方式暴露安培定律的矛盾：數學的與物理的（充電電容悖論）
  • 完整推導馬克士威的修補：位移電流項 \(\mu_0\varepsilon_0\,\partial\vec E/\partial t\)

馬克士威（James Clerk Maxwell，1831–1879，蘇格蘭人， 法拉第發現感應那年出生）從未做過電磁實驗—— 他的儀器是紙筆。1860 年代，他著手把法拉第的力線直覺 翻譯成嚴格數學，卻在整理安培定律時發現： 這條定律病了。

1. 完整推導：連續方程式

電荷守恆的微分形式

電荷守恆（第 2 課）的局部版本：任何體積 \(V\) 內 電荷的減少率，必須等於流出邊界的電流： \[ -\frac{d}{dt}\iiint_V\rho\,dV = \oiint_{\partial V}\vec J\cdot\hat n\,dA \] 右邊用散度定理（第 7 課）化為體積分， 左邊把時間導數移入積分： \[ -\iiint_V\frac{\partial\rho}{\partial t}\,dV = \iiint_V\nabla\cdot\vec J\,dV \] 對任意體積成立 ⟹ 逐點相等： \[ \nabla\cdot\vec J + \frac{\partial\rho}{\partial t} = 0 \] 這就是連續方程式——電荷守恆的微分形式。 穩態（\(\partial\rho/\partial t=0\)）時退化為 \(\nabla\cdot\vec J=0\)（第 18 課節點定律的場版本）。

2. 矛盾（數學版）：取散度

兩行暴露病灶

對安培定律 \(\nabla\times\vec B=\mu_0\vec J\)（第 22 課） 兩邊取散度。左邊：旋度必無散 （第 8 課完整驗證的恆等式）： \[ \nabla\cdot(\nabla\times\vec B) = 0 \] 右邊：\(\mu_0\,\nabla\cdot\vec J\)。所以安培定律強迫 \[ \nabla\cdot\vec J = 0 \quad\text{恆成立} \] 但連續方程式說 \(\nabla\cdot\vec J = -\partial\rho/\partial t\)——只要任何地方的 電荷密度隨時間變化（例如電容充電、電池放電、 天線振盪——幾乎所有有趣的情形）， \(\nabla\cdot\vec J\neq0\)，安培定律就與電荷守恆 正面衝突。第 22 課的「暫定版」警語在此兌現： 這條定律只在穩態成立，不是普適定律。

3. 矛盾（物理版）：充電電容悖論

同一個迴路，兩個曲面，兩個答案

電流 \(I\) 正在為平行板電容（第 14 課）充電。 取一個環繞導線的安培迴路 \(C\)，套用安培定律 \(\oint\vec B\cdot d\vec\ell=\mu_0I_{\text{穿}}\)。 關鍵：\(I_{\text{穿}}\) 是「穿過以 \(C\) 為邊界的 曲面」的電流——但以 \(C\) 為邊界的曲面有 無限多個！
曲面一（平面，被導線刺穿）： \(I_{\text{穿}}=I\)。
曲面二（氣球狀，鼓進兩板之間、 不碰導線）：兩板間沒有電荷流動， \(I_{\text{穿}}=0\)。
同一個 \(\oint\vec B\cdot d\vec\ell\)， 兩個曲面給出 \(\mu_0I\) 與 \(0\) 兩個答案—— 定律自相矛盾。（斯托克斯定理保證答案必須與 曲面選擇無關，第 8 課——所以錯的不是數學， 是這條物理定律不完整。）

4. 馬克士威的修補：位移電流

完整推導：讓散度自動歸零

策略：在安培定律右邊補一項 \(\vec X\)， 使兩邊散度恆等。需要： \[ \nabla\cdot(\mu_0\vec J + \vec X) = 0 \quad\Longleftrightarrow\quad \nabla\cdot\vec X = -\mu_0\nabla\cdot\vec J = \mu_0\frac{\partial\rho}{\partial t} \] （最後一步用連續方程式。）現在用高斯定律（第 12 課） 把 \(\rho\) 換成場：\(\rho=\varepsilon_0\nabla\cdot\vec E\)： \[ \nabla\cdot\vec X = \mu_0\varepsilon_0\frac{\partial}{\partial t}(\nabla\cdot\vec E) = \nabla\cdot\left(\mu_0\varepsilon_0\frac{\partial\vec E}{\partial t}\right) \] 最自然的選擇（馬克士威的選擇）： \[ \vec X = \mu_0\varepsilon_0\frac{\partial\vec E}{\partial t} \] 修補後的安培—馬克士威定律： \[ \nabla\times\vec B = \mu_0\vec J + \mu_0\varepsilon_0\frac{\partial\vec E}{\partial t} \] 新項稱為位移電流（密度 \(\vec J_d=\varepsilon_0\,\partial\vec E/\partial t\)）—— 名字裡有「電流」，實際上沒有任何電荷在移動： 是變化的電場像電流一樣產生磁場。 第 29 課對稱性缺的那一角，被邏輯（不是美學）補上了： 磁變生電（法拉第）、電變生磁（馬克士威）， 記分板四格全滿。

驗算：悖論消解

回到充電電容。曲面二（兩板之間）：傳導電流為零， 但兩板間的電場正在增強（電荷持續堆上兩板， \(E=\sigma/\varepsilon_0\) 隨之增加，第 14 課）。 位移電流：板間 \(E=Q/\varepsilon_0A\)， \[ I_d = \varepsilon_0\frac{\partial E}{\partial t}\times A = \varepsilon_0\cdot\frac{\dot Q}{\varepsilon_0 A}\cdot A = \dot Q = I \] 位移電流恰好等於導線裡的傳導電流——曲面一給 \(\mu_0I\)（全是傳導），曲面二給 \(\mu_0I\) （全是位移），答案一致，悖論消失。 電流「在電容的空隙中以電場變化的形式接力」， 迴路重新閉合。

5. 這一步的份量

值得停下來體會這一步的性質：位移電流不是 由任何實驗發現的——1860 年代沒有任何儀器測得到 它產生的微弱磁場。它是被邏輯一致性 （電荷守恆＋既有定律）逼出來的純理論產物。 物理學史上，用紙筆修改定律、然後預言全新現象的 先例極少——這是最壯麗的一次。修補的直接後果 （下一課推導）：電與磁可以在真空中互相拋接、 脫離一切電荷與導線自行傳播——而它傳播的速度， 將是整門課最大的驚奇。

檢查你的理解

  1. 能完整推導連續方程式 \(\nabla\cdot\vec J+\partial\rho/\partial t=0\)
  2. 能用「取散度」兩行暴露安培定律與電荷守恆的矛盾
  3. 能完整陳述充電電容悖論：兩個曲面如何給出不同答案
  4. 能完整推導位移電流項，並驗算它讓電容悖論消解
