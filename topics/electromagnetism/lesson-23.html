模組 E｜靜磁學：電與磁的第一次牽手 · 第 23 課

磁場沒有源頭：磁單極之謎

學習目標

  • 理解「剖半的磁鐵」實驗事實，以及它與電的根本不對稱
  • 掌握 \(\nabla\cdot\vec B=0\) 的陳述、幾何含義與數學佐證
  • 誠實面對磁單極問題：為什麼理論歡迎它、實驗卻找不到它

電場的散度方程式（第 12 課）說：電荷是電場的源頭。 磁場的對應方程式是什麼？磁的「電荷」——單獨的 N 極 或 S 極——存在嗎？這一課的答案簡短而深刻： 從來沒有人找到過。

1. 剖半的磁鐵

拿一根條形磁鐵，N 極在一端、S 極在另一端。 從中間鋸開，期待得到「一根純 N」與「一根純 S」？ 實驗結果：得到兩根完整的小磁鐵， 各自有自己的 N 極與 S 極。再剖，再得四根； 一路剖到原子尺度，永遠是完整的偶極。 與電的對比再鮮明不過：正負電荷可以輕易分開 （摩擦就行，第 2 課），N 與 S 卻像影子一樣 永不分離。

安培分子電流的解釋

第 21 課的「電流環＝小磁鐵」給出了漂亮的解釋： 如果磁鐵的磁性來自物質內部無數微觀環形電流 （安培的假說），那麼「剖半得到兩個完整磁鐵」 就完全自然——你只是把一疊小電流環分成兩疊， 每一疊仍然是一疊電流環。N 極與 S 極不是兩種 「磁荷」，只是同一個電流環的兩面—— 問「能不能把 N 面和 S 面分開」，就像問能不能 把硬幣的正面和反面分開。

2. \(\nabla\cdot\vec B=0\)：第二條馬克士威方程式

陳述與佐證

把「沒有磁荷」寫成場方程式： \[ \nabla\cdot\vec B = 0 \qquad\Longleftrightarrow\qquad \oiint_S\vec B\cdot\hat n\,dA = 0 \] 磁場穿過任何封閉曲面的淨通量恆為零—— 磁力線沒有起點、沒有終點，只能閉合成圈 （或延伸到無窮遠）。多少力線進入一個區域， 就有多少力線離開。

數學上的佐證來自第 8 課的恆等式：由必歐—沙伐定律 產生的磁場，可以證明總能寫成某個向量場的旋度 （\(\vec B=\nabla\times\vec A\)，\(\vec A\) 稱為向量位； 完整證明需要對必歐—沙伐積分做向量分析改寫， 技術細節誠實略過），而旋度必無散 （\(\nabla\cdot(\nabla\times\vec A)=0\)，第 8 課 完整驗證過）——電流產生的磁場自動 滿足無散。這條方程式的地位特殊：它是馬克士威 方程組中唯一「右邊是零」的——沒有源項， 因為找不到源。

3. 電與磁的記分板（到目前為止）

把已推導的場方程式並排，看見一種「跛腳的對稱」：

四格記分板

\[ \begin{aligned} &\text{散度：}\quad \nabla\cdot\vec E = \frac{\rho}{\varepsilon_0} &&\qquad \nabla\cdot\vec B = 0\\ &\text{旋度：}\quad \nabla\times\vec E = 0 &&\qquad \nabla\times\vec B = \mu_0\vec J \end{aligned} \] 電有散度源（電荷）而無旋度源；磁有旋度源（電流） 而無散度源——恰好互補、又不對稱。這個表格還會 再變兩次：模組 F 給 \(\nabla\times\vec E\) 加上時變項 （法拉第），模組 G 給 \(\nabla\times\vec B\) 加上 時變項（馬克士威）——每加一項，表格就更對稱一分。 記住它現在的模樣，之後的每一步都是對照著它前進的。

4. 誠實的懸念：磁單極子

理論歡迎它，實驗找不到它

\(\nabla\cdot\vec B=0\) 是一條實驗陳述—— 它說的不是「磁單極不可能存在」，而是「至今從未 觀察到」。有趣的是，二十世紀的理論物理反而 歡迎磁單極：狄拉克 1931 年證明， 只要宇宙中存在一顆磁單極，就能解釋為什麼電荷 是量子化的（「原子論」主題第 1 課密立坎測到的 電荷都是 \(e\) 的整數倍——至今沒有更好的第一原理 解釋）；後來的大一統理論更是普遍預言 磁單極存在（這些內容屬於「標準模型」主題第 58 課 層級的物理，此處只誠實指出去向）。實驗上， 數十年的專門搜尋（海水、月岩、宇宙線、對撞機） 一無所獲。所以這條最乾淨的方程式，同時是 物理學一個著名的未解問題：右邊那個零， 是宇宙的定律，還是只是我們還沒找到？ 這門課依慣例把懸念原樣留下。
