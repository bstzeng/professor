奧伯斯悖論：為什麼夜空是黑的

學習目標

  • 完整推導「無限、永恆、靜態宇宙」預測的夜空亮度發散積分
  • 理解為什麼「夜空是黑的」這個日常事實，其實是深刻的宇宙學證據
  • 認識悖論的兩個現代解答：宇宙年齡有限與宇宙膨脹

模組 F 要處理這門課最後、也最大的問題：宇宙從哪裡來、怎麼演化、往哪裡去。 入口是一個看似天真的問題——晚上抬頭，天空為什麼是黑的？ 這個問題困擾了天文學家數百年，而它的答案直接指向大霹靂。

1. 悖論的設定

假設宇宙是無限大、永恆存在、靜態不變的（這是二十世紀之前 大多數人的預設圖像），恆星以平均數密度 \(n\)（每單位體積的恆星數） 均勻分布，每顆恆星的光度為 \(L\)。這三個假設看起來都很「自然」， 但接下來的推導會顯示它們合起來和「夜空是黑的」直接矛盾。

2. 完整推導：亮度積分發散

以地球為中心，考慮半徑 \(r\)、厚度 \(dr\) 的球殼。球殼體積：

\[ dV = 4\pi r^2\,dr \]

球殼內的恆星數：

\[ dN = n\,dV = 4\pi n r^2\,dr \]

每顆距離 \(r\) 的恆星，依反平方定律（第 3 課）貢獻的通量是 \(L/(4\pi r^2)\)。 整個球殼貢獻的總通量：

關鍵一步：距離消去

\[ dF = \frac{L}{4\pi r^2}\cdot 4\pi n r^2\,dr = nL\,dr \]

反平方定律的 \(r^{-2}\) 衰減，恰好被球殼恆星數的 \(r^2\) 成長完全抵消—— 每一層球殼貢獻的亮度都一樣，和距離無關。

把所有球殼從 \(r=0\) 積分到無限遠：

\[ F_{\text{total}} = \int_0^{\infty} nL\,dr = nL\int_0^{\infty}dr \longrightarrow \infty \]

積分發散——無限、永恆、靜態的宇宙預測夜空應該無限亮。 即使考慮近處恆星會遮住遠處恆星（恆星有有限的角大小，遠處的光會被擋住）， 修正後的結論也只是從「無限亮」變成「每一條視線最終都會落在某顆恆星的表面上」—— 整個天空應該像太陽表面一樣亮，晚上和白天沒有區別。這顯然與事實不符， 矛盾就是奧伯斯悖論。

3. 現代解答

三個假設至少有一個是錯的。現代宇宙學的答案是：兩個都錯。

  • 宇宙年齡有限（主要原因）：宇宙有一個起點（約 138 億年前， 第 34 課），光速又有限（第 26 課的光行時概念），所以我們只能看到 距離 \(r\lesssim ct_0\) 以內的天體發出的光。上面的積分上限不是無限大， 而是被「可觀測宇宙」的尺度截斷： \[ F_{\text{total}} \approx nL\cdot ct_0 \quad(\text{有限值}) \] 代入實際的恆星密度與光度，這個值遠低於白天的亮度——夜空因此是黑的。
  • 宇宙膨脹（次要貢獻）：第 33 課會介紹，宇宙在膨脹， 遠方天體的光被紅移到更低的能量、更低的到達率，進一步壓低遠方球殼的 亮度貢獻。定量分析顯示這個效應是次要的——就算宇宙不膨脹， 光是年齡有限就足以讓夜空是黑的。

一個日常觀察的宇宙學份量

「夜空是黑的」因此不是一句廢話，而是一個貨真價實的宇宙學觀測事實： 它直接排除了「無限、永恆、靜態」的宇宙圖像，指向宇宙有一個有限的過去。 你每天晚上抬頭，都在親眼目睹大霹靂宇宙觀最直接的證據之一。

檢查你的理解

  1. 能完整推導球殼亮度貢獻 \(dF=nL\,dr\)，說明 \(r^2\) 是怎麼消掉的
  2. 能說明「每條視線最終落在恆星表面」這個修正版結論怎麼來的
  3. 能說出悖論的兩個現代解答，以及哪一個是主要原因
  4. 能解釋為什麼「夜空是黑的」可以當成宇宙有限年齡的證據
