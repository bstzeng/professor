模組 A｜觀測基礎：我們怎麼「看」宇宙 · 第 5 課

測光與濾鏡：顏色指數與觀測資料怎麼來的

學習目標

  • 認識標準測光系統（UBVRI）與顏色指數的定義
  • 完整推導卜瓦松分布的均值與變異數相等這個關鍵性質
  • 從光子計數統計推導訊噪比公式，理解「觀測資料的誤差」從哪裡來
  • 認識熱星等改正的概念，理解為什麼單一濾鏡量到的星等不等於總光度

模組 A 前四課建立了「怎麼收集光」「怎麼量化亮度」「光譜線代表什麼」。這一課要處理最後一塊、 也是最貼近實務的部分：望遠鏡實際量到的每一個數字，背後有什麼系統性的定義（濾鏡、顏色指數）， 又帶有多大的隨機誤差（光子計數統計）。這一課建立的訊噪比工具，會貫穿之後每一課牽涉到 「這個訊號可不可信」的討論。

1. 標準測光系統：UBVRI

望遠鏡很少只用「總光量」這種單一數字，而是搭配一組標準化的濾鏡，各自只讓某個波段的光通過， 分別量測星等。最常用的 Johnson-Cousins 系統包含：

  • U（ultraviolet）：中心波長約 365 nm，近紫外
  • B（blue）：中心波長約 445 nm
  • V（visual）：中心波長約 551 nm，接近人眼最敏感的區域
  • R（red）：中心波長約 658 nm
  • I（infrared）：中心波長約 806 nm，近紅外

每個濾鏡各自量出一個視星等（第 3 課定義的 \(m\)），例如 \(m_B\)、\(m_V\)。 重點不是單一濾鏡的星等本身，而是不同濾鏡星等的差——這就是顏色指數。

2. 顏色指數

定義顏色指數（例如 \(B\!-\!V\)）為兩個濾鏡星等的差：

\[ (B-V) \equiv m_B - m_V \]

用第 3 課的 Pogson 公式展開，會發現 \(B-V\) 直接等於兩個波段通量比的對數 （常數項在相減時大部分抵消，只留下和恆星本身光譜形狀有關的部分）， 不受距離影響——因為距離造成的通量衰減（反平方定律）對所有波段是同一個因子 \(1/4\pi d^2\)， 相減時完全消掉。這是顏色指數的關鍵優勢：不需要知道距離，就能得到恆星的內稟資訊。 第 8 課會證明 \(B\!-\!V\) 和恆星表面溫度之間有明確的物理關係（顏色愈藍代表溫度愈高）。

3. 光子計數統計：卜瓦松分布（完整推導）

望遠鏡的偵測器（CCD）本質上是在數光子。光子到達是一個隨機過程，在固定觀測時間內 收到的光子數 \(N\) 服從卜瓦松分布：

\[ P(N=k) = \frac{\lambda^k e^{-\lambda}}{k!}, \qquad k=0,1,2,\dots \]

（\(\lambda\) 是平均光子數。）這一節要證明一個對觀測天文學極其重要的性質： 卜瓦松分布的變異數等於它的均值，也就是 \(\text{Var}(N)=\lambda\)。

先算均值：

\[ E[N] = \sum_{k=0}^{\infty} k\cdot\frac{\lambda^k e^{-\lambda}}{k!} = \lambda e^{-\lambda}\sum_{k=1}^{\infty}\frac{\lambda^{k-1}}{(k-1)!} = \lambda e^{-\lambda}\cdot e^{\lambda} = \lambda \]

（第一項 \(k=0\) 不貢獻，把 \(k\) 消掉一個、指標平移後，剩下的求和正好是 \(e^\lambda\) 的泰勒展開。） 接著算 \(E[N(N-1)]\)，用同樣的技巧：

\[ E[N(N-1)] = \sum_{k=0}^{\infty} k(k-1)\frac{\lambda^k e^{-\lambda}}{k!} = \lambda^2 e^{-\lambda}\sum_{k=2}^{\infty}\frac{\lambda^{k-2}}{(k-2)!} = \lambda^2 e^{-\lambda}\cdot e^{\lambda} = \lambda^2 \]

因為 \(E[N(N-1)] = E[N^2]-E[N]\)，所以 \(E[N^2] = \lambda^2+\lambda\)。變異數：

卜瓦松分布的均值等於變異數

\[ \text{Var}(N) = E[N^2] - (E[N])^2 = (\lambda^2+\lambda) - \lambda^2 = \lambda \]

均值和變異數都等於 \(\lambda\)。這代表光子計數的標準差（雜訊大小）就是 \(\sqrt{\lambda}\)——收到的光子數愈多，絕對雜訊愈大，但相對雜訊 \(\sqrt{\lambda}/\lambda = 1/\sqrt{\lambda}\) 反而愈小。這就是為什麼拉長曝光時間 （收集更多光子）能提高資料品質：不是因為雜訊消失，而是相對雜訊按 \(1/\sqrt{\lambda}\) 下降。

4. 訊噪比公式

實際觀測中，望遠鏡收到的光子同時包含目標天體的訊號 \(N_s\) 與背景天光的訊號 \(N_b\) （大氣輝光、雜散光等），兩者都是獨立的卜瓦松過程，總計數 \(N_{\text{total}}=N_s+N_b\)。 因為兩個獨立卜瓦松過程的變異數可以直接相加（獨立隨機變數的變異數具有可加性）：

\[ \text{Var}(N_{\text{total}}) = \text{Var}(N_s) + \text{Var}(N_b) = N_s + N_b \]

扣除背景估計值後，目標訊號本身是 \(N_s\)，但雜訊仍然來自兩者共同貢獻。 定義訊噪比（signal-to-noise ratio）為訊號除以雜訊標準差：

光子計數訊噪比公式

\[ \text{SNR} = \frac{N_s}{\sqrt{N_s + N_b}} \]

（這裡先忽略偵測器本身的讀出雜訊、暗電流等額外雜訊來源，只考慮光子計數本身的統計極限， 稱為「光子雜訊極限」。）這個公式說明兩件事：目標愈暗（\(N_s\) 小）或背景愈亮 （\(N_b\) 大，例如滿月夜晚或光害嚴重的城市），訊噪比就愈差； 要提高訊噪比，可以拉長曝光時間讓 \(N_s, N_b\) 同時等比例增加——由公式可以驗證， 當 \(N_s \gg N_b\) 時 \(\text{SNR}\approx\sqrt{N_s}\)，訊噪比隨曝光時間的平方根成長， 這是深空攝影需要長時間曝光的數學原因。

5. 熱星等改正：從單一濾鏡到總光度

第 3 課定義的星等（不論哪個濾鏡）只涵蓋該濾鏡的波段範圍，不是恆星輻射的全部能量。 定義熱星等（bolometric magnitude）\(m_{\text{bol}}\) 對應恆星在所有波段 輻射的總光度，兩者的差稱為熱改正（bolometric correction）：

\[ \text{BC} \equiv m_{\text{bol}} - m_V \]

熱改正的大小取決於恆星的光譜能量分布——溫度極高或極低的恆星，大部分能量會落在 \(V\) 波段之外（分別偏向紫外或紅外），熱改正的絕對值就比較大。這個修正需要知道完整的 黑體輻射光譜形狀才能精確計算，是第 7、8 課的內容。

檢查你的理解

  1. 能說出顏色指數 \(B-V\) 為什麼不受距離影響——這用到了反平方定律的哪個性質？
  2. 能重新推導一次卜瓦松分布 \(E[N]=\lambda\) 和 \(\text{Var}(N)=\lambda\)
  3. 能推導訊噪比公式 \(\text{SNR}=N_s/\sqrt{N_s+N_b}\)，並說明「獨立隨機變數變異數可加」這個性質用在哪裡
  4. 能解釋為什麼拉長曝光時間能提高訊噪比，且訊噪比是按曝光時間的平方根成長

模組 A 小結

五堂課建立了整套觀測工具箱：電磁波與繞射極限決定了「能看多細」（第 1 課）， 望遠鏡設計把這個極限實際做成儀器（第 2 課），星等系統把亮度量化成可比較的數字（第 3 課）， 光譜學把「線的位置與位移」轉換成化學組成與速度資訊（第 4 課），測光與統計則處理 「這些量測數字有多可信」（第 5 課）。模組 B 要開始正式回答你最初問的問題： 怎麼知道一顆星的距離、大小、質量——每一個測量方法都會直接用到這一模組建立的工具。
